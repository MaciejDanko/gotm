INTERNAL: The gradient of the log likelihood function

Description

INTERNAL: The gradient of the log likelihood function

Usage

hopit_derivLL(parameters = model$coef, model, collapse = TRUE,
  use_weights = model$use.weights, negative = FALSE)

Arguments

parameters

model parameters (optional). If not delivered then taken from model$coef.

model

hopit object.

collapse

a logical indicating if to sum individual LL contributions.

use_weights

a logical indicating if to use model weights.

negative

a logical indicating if the function should return negative LL or not.
