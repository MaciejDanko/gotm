hopit_fitter {hopit}R Documentation

INTERNAL: Fit hopit model given a starting parameters

Description

Fit the model.

Usage

hopit_fitter(model, start = model$start,
  use_weights = model$use.weights)

Arguments

model

a hopit object.

start

starting parameters.

use_weights

a logical indicating if to use model weights.
