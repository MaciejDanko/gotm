Plot the log likelihood profile for a profile.hopit object

Description

Plot method for profile.hopit object.

Usage

## S3 method for class 'profile.hopit'
plot(x, ..., ylim = NULL, relative = FALSE, leg.cex = 0.85, leg.col = 'blue4')

Arguments

x

an profile.hopit object.

...

arguments to be passed to plot() function (see par).

ylim

see plot.

relative

logical indicating if ylim on each panel should be the same (TRUE) or not (FALSE).

leg.cex

character expansion factor relative to current par("cex") (see legend).

leg.col

the color used for the legend text.
