hopit_ExtractParameters {hopit}R Documentation

INTERNAL: Extract model parameters as a list

Description

Extract model parameters as a list.

Usage

hopit_ExtractParameters(model, parameters, parcount = model$parcount)

Arguments

model

hopit object.

parameters

model parameters (optional). If not delivered then taken from model$coef.

parcount

vector with parameter counts for latent, lambda, and gamma.
